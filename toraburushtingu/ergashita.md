# エラーが発生した場合

VistaUpdaterでエラーが発生した場合は、以下の方法を試してみてください。

## VistaUpdaterを最新版に更新する

VistaUpdaterの古いバージョンを実行している場合、最新版に更新するとバグが修正される可能性があります。

1. タスクマネージャーを開き(Ctrl+Alt+Delete->タスクマネージャー)、上部のファイル->新しいタスクの実行をクリックします。
2. 「iexplore」と入れて「OK」をクリックします。
3. [http://vistaupdater.net](http://vistaupdater.net)へアクセスし、VistaUpdaterをダウンロードして実行します。

## VistaUpdater修復ツールを試す

修復ツールを使用すると、VistaUpdaterを最新版に更新したり元の状態に戻したりできます。

長い間メンテナンスをしてしまい、申し訳ございませんでした。

1. タスクマネージャーを開き(Ctrl+Alt+Delete->タスクマネージャー)、上部のファイル->新しいタスクの実行をクリックします。
2. 「iexplore」と入れて「OK」をクリックします。
3. [http://vistaupdater.net/tools](http://vistaupdater.net/tools)へアクセスし、VistaUpdater 修復ツールをダウンロードして実行します。
4. 修復を開始をクリックします。
5. 自分のしたい処理をクリックします。
6. 処理が終了するまで待ち、処理が完了してWindowsが正常に起動して使えることを確認します。

## システムの復元を試す

システムの復元ツールで、システムを元の状態に戻すことができます。

1. Ctrl+Alt+Deleteを押して、右下の電源マークから再起動をクリックします。
2. 起動するときにF8を連打して、詳細ブートオプションが起動するまで連打します。
3. 詳細ブートオプションが表示されたら、コンピューターの修復を十字キーで選択してEnterで決定します。
4. システム回復オプションが表示されたら、次へをクリックします。
5. ユーザー名の欄でユーザー名を選び、パスワードの欄でパスワードを入力します。入力したら、OKをクリックします。
6. システムの復元をクリックします。
7. システムの復元が表示されたら、次へをクリックします。
8. 任意の復元ポイントを選択して、次へをクリックします。

{% hint style="info" %}
他の復元ポイントを表示にチェックを入れると、古い復元ポイントが表示されます。
{% endhint %}

9. 完了をクリックします。
10. 処理が完了するまで待ち、完了したらWindowsが正常に起動して使えることを確認します。
